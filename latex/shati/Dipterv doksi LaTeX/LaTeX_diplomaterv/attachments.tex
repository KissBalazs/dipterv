\section{Forrásfájlok}\label{sect:SourceCodeAttachment}
%----------------------------------------------------------------------------
A Diplomaterv Portálon (\url{https://diplomaterv.vik.bme.hu/}) a szakdolgozatom mellékleteként feltöltött \textit{melleklet\_portal.zip}, illetve a bekötött szakdolgozathoz mellékelt DVD lemezen lévõ \textit{melleklet\_dvd.zip} tömörített állományok \textit{M\_1\_Forrasfajlok} mappájában találhatóak a mobilalkalmazás végleges verziójának git verziókezelõ használatára alkalmas forrásfájljai.

A dolgozat \sectref{DevelopmentInfrastructureChapter}. fejezetében ismertetettem annak az infrastruktúrának\linebreak a létrehozását, amelyben az alkalmazás fordítható. A \textit{Visual Studio Solution} fájl az\linebreak \textit{M\_1\_Forrasfajlok\texttt{\textbackslash}SmartActive\texttt{\textbackslash}SmartActive.sln} útvonalon található. Fordítási nehézségek esetén az \textit{M\_1\_Forrasfajlok\texttt{\textbackslash}README.md} szöveges fájl tartalma ad útmutatást.

\section{Telepíthetõ alkalmazáscsomagok}\label{sect:InstallerAttachment}
%----------------------------------------------------------------------------
A bekötött szakdolgozathoz mellékelt DVD lemezen lévõ \textit{melleklet\_dvd.zip} tömörített állomány \textit{M\_2\_Telepitheto\_alkalmazascsomagok} mappájában találhatóak azok a fájlok, amik telepíthetõek egy megfelelõen konfigurált Android, Windows Phone 8.1 vagy Windows 10 Mobile operációs rendszerû telefonra. iOS operációs rendszerhez nem készült ilyen alkalmazáscsomag, mert nem rendelkezem a lehetõséget nyújtó Apple Developer Enterprise fejlesztõi licenc-szel.
Ez a melléklet a Diplomaterv Portál (\url{https://diplomaterv.vik.bme.hu/}) maximális megengedett fájlméretét meghaladja, ezért oda nem került feltöltésre.

További információk a csomagok telepítéséhez az alábbi weboldalakon találhatók:
\begin{itemize}
	\item Android: \texttt{http://www.cnet.com/how-to/how-to-install-apps-outside-of-\linebreak  google-play/}
	\item Windows Phone 8.1: \texttt{https://msdn.microsoft.com/en-us/library/windows/\linebreak apps/xaml/dn832613.aspx}
	\item Windows 10 Mobile: \texttt{https://msdn.microsoft.com/windows/uwp/packaging/\linebreak  install-universal-windows-apps-with-the-winappdeploycmd-tool}
\end{itemize}
